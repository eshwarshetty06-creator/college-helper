# SIT-TUMKUR Campus Helper System

A comprehensive campus management system with features for students and administrators.

## Features

- **Classroom Management** - View and manage classroom status
- **Issue Reporting** - Students can report issues, admins can resolve them
- **Announcements** - Admins post announcements, students view them
- **Lost & Found** - Report and track lost items
- **Study Room Booking** - Book study rooms with admin approval

## Tech Stack

- HTML, CSS, JavaScript
- Firebase Authentication
- Firebase Firestore Database

## Deployment

This project is configured for Netlify deployment.

## Firebase Configuration

The Firebase configuration is already set up in the JavaScript files. Make sure your Firebase project is properly configured with:
- Authentication enabled (Email/Password)
- Firestore Database enabled
- Proper security rules set
